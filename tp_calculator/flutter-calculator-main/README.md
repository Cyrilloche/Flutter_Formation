# Calculator

TP d'initiation à Flutter et au langage Dart.

## Commandes utiles

- Installation des dépendances

```BASH
flutter pub get
```

- Lancement de l'application

```BASH
flutter run
```

- Mise à jour de Flutter

```BASH
flutter upgrade
```

- Mise à jour des dépendances

```BASH
flutter pub upgrade
```
